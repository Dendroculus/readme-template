# Command Tables

## Discord

| Command | Category | Description |
|---|---|---|
| `/profile [user]` | Progression | Display a profile card |
| `/leaderboard` | Progression | Show ranked members |
| `/help` | General | List available commands |

## CLI

| Command | Description |
|---|---|
| `tool scan PATH` | Scan a project |
| `tool apply PATH` | Apply changes |
| `tool restore BACKUP` | Restore a backup |

Keep optional parameters in brackets and required parameters in angle brackets or plain names consistently.
