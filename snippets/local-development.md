# Local Development Snippets

## Python

```bash
python -m venv .venv
```

```powershell
# Windows PowerShell
.venv\Scripts\Activate.ps1
```

```bash
# macOS / Linux
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
```

## Node.js

```bash
npm install
npm run dev
```

## Copy Environment Files

```powershell
Copy-Item .env.example .env
```

```bash
cp .env.example .env
```
